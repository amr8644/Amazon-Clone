## Run Locally

Clone the project

```bash
  git clone https://github.com/amr8644/Amazon-Clone.git
```

Go to the project directory

```bash
  cd Amazon-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Tech Stack

**Client:** Next JS,TailwindCSS

**Server:** Node, Next-Auth, Prisma, PlanetScale
